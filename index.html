Wirless Sensor Network for Outdoor Security

The security system make use of wirless sensor network that collects data from different sensors which is powered by microcontroller consist of Temperature sensor,PIR sensor ,camera.The iot based system detects the tempature ,motion using pir sensor and temperature sensor and sends all the data to the adafruit which display your data in real time and make the project internet-connected which reads sensor data .It also handle and visualize multiple feeds of data.The system uses GPy.
